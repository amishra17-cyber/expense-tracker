Expense Tracker :
A simple web app I built to track daily income and expenses.
I made this project to practice full stack web development
using Python, Flask and SQLite.

What this app does :
- You can add your daily income and expenses
- It shows your total balance, total income and total expense
- All your transactions are saved in a database
- You can delete any transaction anytime
- Transactions are sorted by date so latest ones show first

Technologies I used :
. HTML and CSS for the frontend design
. Python and Flask for the backend
. SQLite to store the transaction data

How to run this project :
First make sure you have Python installed on your system.
clone the repository:
Then install Flask by running this command : pip install flask
Then run the app : python app.py

Then open your browser and go to:
http://localhost:5000

What I learned :
While building this project I learned how to connect
a frontend form to a Python backend, how to store and
retrieve data from a database, and how Flask routing works.
